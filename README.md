# BMI Calculator

A responsive and modern BMI Calculator built using HTML, CSS, and JavaScript.

## Features
- Calculate BMI instantly
- Input validation
- Displays BMI category
- Modern glassmorphism UI
- Smooth hover animations
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)

## BMI Categories
- Underweight
- Normal Weight
- Overweight
- Obese
